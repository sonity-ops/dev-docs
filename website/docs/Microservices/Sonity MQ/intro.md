# Intro

Sonity MQ is a queing system built on top of Bull MQ

![Sonity MQ Flow](/img/docs/sonity-mq.png)

This allows us to execute tasks in the background:
- Sending Emails
- Data Exports
- Data Imports (Maybe soon)


## Tech Stack
- Express (Typescript)
- Sequelize (ORM)
- Postgres (Database)
